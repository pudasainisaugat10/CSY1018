About my Design, Colors and Fonts

Web design is the process of creating a website. Web design contains several different aspects, including web page layout, content, text and many others. Generally website are created using a marking language called HTML. Web designers build their webpage using HTML tags that defines the content and data of each pages. Similarly my web page design is in simple and attractive form. I used background-image to make my web page more attractive. Inserting background-image in web page adds the visual interest to a page and helps us to achieve the design we are looking for. I used social media links in my webpage. It make it easier to the audience to find the other content through the website. I used grid-columns to achieve pixel perfect website layouts. To make my web page attractive I used many different tags and codes like display inline-block, float left, position fixed and other tags too, it helps to achieve the perfect website layout design.

Colors are the first things people notice when they visit our website. Colors scheme can make a huge impact on colors both style and consistency. Consistency is actually one of the fundamental values in creating a color scheme for the website. Web designers use HTML and CSS tags to insert different color in webpage. Similarly for my web page I used different colors like white, black, dark and light color, red and marron. For the designing web page I used white and black color more than other colors. For main text mainly I used white and black color. I used dark color for background image, light color and white color for footer, red for page layout and marron for the text. Using the colors made my website more attractive.

Fonts play a major role in shaping the brand language and website design. Fonts make the website unique, perfect and attractive. Basically fonts are used to format the text on website. People are more familiar with Arial, Helvetica and Times New Roman for designing the website. Especially font choice is important for branding and it helps convey brand values and personality. Fonts also impact how effectively people can view and navigate your website. Similarly for my web page I used different fonts for the different headings and different font size for different text. I used normal font style for my all three pages but for the contact.html and cv.html page I used italic and oblique font style. Font size are different for all the pages. For the all text I used text-align justify to make it more perfect and attractive and putting the perfect font style and font size helps to achieve the perfect web page.
